# Tripago

## Prequisite

node.js

## Installation

### tripago

cd into triapgo/, in package.json, cut package "animation..." temporariliy

```
npm install
```
paste back "animation..."

## tripago-server

cd into tripago-server/

```
npm install
```

## Run

### Step 1 : tripago-server

cd into tripago-server/

```
npm start
```

### Step 2 : tripago

cd into tripago/

```
npm start
```

### Step 3 : Google Chrome

```
localhost:8080
```
